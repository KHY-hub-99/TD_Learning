# GridWorld TD Learning Simulation

이 프로젝트는 **TD Learning**을 사용하여, 동적으로 벽이 생성되는 $4 \times 4$ 그리드 월드(Grid World) 환경에서 에이전트의 상태 가치(State Value)를 학습하는 시뮬레이션입니다.

---

## 1. 개요

에이전트는 **(0, 0)**에서 출발하여 **(3, 3)** 목적지에 도달하는 것을 목표로 합니다. 무작위 정책(Random Policy)을 수행하며, 매 에피소드마다 변화하는 장애물 배치 속에서 각 지점의 기대 보상(상태 가치)을 효과적으로 추정합니다.

## 2. 환경 구성 (GridWorld)

- **맵 크기**: $4 \times 4$ 정사각 그리드
- **주요 지점**: 시작점 (0, 0), 목표점 (3, 3)
- **보상 체계**: 이동 1회당 **-1**의 보상 (최단 경로 유도)
- **장애물 시스템**:
  - 매 리셋(`reset`) 시 2~3개의 벽이 무작위 위치에 생성
  - 시작/종료 지점 봉쇄 방지 및 외곽 벽 생성 제한 로직 적용
  - 생성된 벽은 양방향 통행 불가 처리

## 3. 학습 알고리즘: TD(0)

에이전트는 아래의 **TD(0) 예측(Prediction)** 공식을 사용해 상태 가치 $V(s)$를 업데이트합니다.

$$V(s) \leftarrow V(s) + \alpha [r + \gamma V(s') - V(s)]$$

- **학습률 ($\alpha$)**: 0.01
- **할인율 ($\gamma$)**: 1.0 (감쇄 없음)
- **반복 횟수**: 10,000 에피소드

## 4. 코드 주요 모듈

| 클래스/함수   | 역할 설명                                                      |
| :------------ | :------------------------------------------------------------- |
| **GridWorld** | 좌표 관리, `step` 함수를 통한 보상 처리 및 무작위 벽 생성 로직 |
| **Agent**     | 상, 하, 좌, 우 4방향 무작위 행동(`select_action`) 선택         |
| **main**      | TD 업데이트 루프 실행 및 1,000회 주기마다 현재 가치 행렬 출력  |

## 5. 실행 및 결과 확인

에이전트가 학습을 진행함에 따라 목표 지점인 (3, 3)에 가까울수록 가치(Value)가 0에 가까워지고, 멀어질수록 음수 값이 커지는 것을 확인할 수 있습니다.

```python
# 실행 방법
python grid_world_td.py
```

- **실시간 모니터링**: 1,000 에피소드마다 현재 막힌 경로의 좌표와 업데이트된 상태 가치 행렬을 출력합니다.
- **학습 특징**: 매번 벽의 위치가 바뀌기 때문에, 최종 결과는 특정 경로가 아닌 전체적인 맵의 기하학적 구조에 따른 기대 보상값을 나타냅니다.
